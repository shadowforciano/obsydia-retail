Install RustDesk

Last updated: Feb 10, 2026

RustDesk is a remote desktop app used for support and access across devices.

You may see a warning that says "You are being scammed" on the RustDesk website. We will only access your computer with your permission to install RustDesk, and you can delete it afterward and change your password. Our goal is to be professional and do a great job.

Download from official sources

Use the official RustDesk site or GitHub releases to avoid fake installers.

  • RustDesk official website
  • RustDesk GitHub releases
  • RustDesk client documentation

Windows (10/11)

Direct download: RustDesk 1.4.5 (Windows 64-bit)

  1. Download the Windows installer (.exe or .msi) from GitHub releases.
  2. Open the file and follow the installer prompts.
  3. Launch RustDesk and allow firewall access if prompted.
  4. Share your ID and one-time password with the person helping you.

macOS

  1. Download the macOS .dmg from GitHub releases.
  2. Open the .dmg and drag RustDesk into Applications.
  3. If macOS blocks it, go to System Settings > Privacy & Security and allow it to open.
  4. Grant Accessibility, Screen Recording, and Input Monitoring permissions when prompted.

Linux

  1. Download the package for your distro (.deb, .rpm, or AppImage) from GitHub releases.
  2. Install it using your software center or package manager.
  3. Launch RustDesk from your applications menu.
  4. Allow any requested permissions for screen control.

Android

  1. Download the Android APK from GitHub releases.
  2. Open the APK and allow installation from this source if prompted.
  3. Open RustDesk and note your ID and password.

iPhone / iPad

  1. RustDesk on the App Store
  2. Install the app and open it.
  3. You can control other devices from iOS, but iOS devices cannot be controlled.

First connection and safety

  1. Only share your RustDesk ID and one-time password with people you trust.
  2. If you did not request help, close the app and do not share your code.
  3. End the session when support is finished.

RustDesk shows a new password each time it starts. This helps prevent unwanted access.
