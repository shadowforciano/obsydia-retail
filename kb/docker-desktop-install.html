Install Docker Desktop

Last updated: Feb 10, 2026

Docker Desktop lets you run containers on Windows and macOS with a simple desktop app.

Download from official sources

Use Docker's official download page and documentation to avoid fake installers.

  • Docker Desktop download page
  • Docker Desktop documentation

Windows (10/11)

Direct download: Docker Desktop Installer (Windows 64-bit)

  1. Download the Windows installer from the official page.
  2. Run the installer and keep the WSL 2 option enabled if prompted.
  3. Restart your PC if the installer asks.
  4. Open Docker Desktop and wait for the engine to show as running.

macOS

  1. Download the correct installer for your Mac (Apple Silicon or Intel).
  2. Open the .dmg and drag Docker Desktop into Applications.
  3. Open Docker Desktop and approve the permissions when prompted.
  4. Wait for the engine status to show as running.

Verify the install (optional)

  1. Open Terminal (macOS) or PowerShell (Windows).
  2. Run: docker --version
  3. Run: docker run hello-world

If you see a success message, Docker Desktop is ready to use.

Linux note

Docker Desktop is for Windows and macOS. On Linux, install Docker Engine using your distro's package manager.
