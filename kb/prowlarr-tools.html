Prowlarr Tools & Indexer Troubleshooting

Last updated: Feb 10, 2026

This guide explains the core tools and how to add or fix indexers in Prowlarr.

Technology stack overview

These tools work together to find content and send it to the right app.

Prowlarr (Indexers)

Central place to manage indexers and push results to Sonarr/Radarr.

Sonarr (TV)

Manages TV series and sends search requests to indexers.

Radarr (Movies)

Manages movies and sends search requests to indexers.

qBittorrent

Download client that receives files from Sonarr/Radarr.

How to add indexers in Prowlarr

Use these steps to add an indexer and test it before saving.

  1. Open Prowlarr and go to Indexers.
  2. Click Add Indexer and choose the provider.
  3. Paste the API key and base URL from the indexer.
  4. Click Test, then Save when it succeeds.

If an indexer fails

Most failures are fixed by switching the base URL or using a different indexer.

  1. If one indexer fails, switch the base URL and test again.
  2. If it still fails, try a different indexer.
  3. If all indexers fail, check that your API key and credentials are correct.
  4. Try again later in case the indexer is temporarily down.
