FAQ

Last updated: Feb 10, 2026

Common questions about the home server system.

What can this system do?

It is a home server with open-source tools that help organize and stream legally obtained content. We do not include content — users decide how they use it.

Can it download movies?

The system includes general-purpose download tools. What the user downloads is entirely their responsibility and must comply with copyright laws.

Does it come with Pirate Bay or similar sites?

No. The system includes open-source management software. We do not provide, recommend, or guarantee access to any specific content or sources.
